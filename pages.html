Borrow This!

My final project for General Assembly: A web application similar to craigslist, Sharegrid, or AirBnB where users can post items for rent and make reservations for those items. In addition users can decide how much money they want to charge, whether to approve reservations, and give reviews for items on the website.

My Bookshelf

Javascript - Node Express - Mongo DB

An application where users can create bookshelves storing information about their favorite books (or books currently reading). Read reviews they have submitted for each book and see other users saved bookshelves.

Wayfarer

Python - Django - PostgreSQL - Bulma

A web application where users can interact with city travel destinations by adding posts and viewing other users' posts sharing travel tips per city along the way. Each user can also navigate their own profile page showing recent posts as well as make comments on posts.
